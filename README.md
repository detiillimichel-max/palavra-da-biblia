# palavra-da-biblia
40 perguntas bíblicas (AT e NT) — cada rodada sorteia 20 aleatórias
6 categorias para filtrar: Todas, Antigo Test., Novo Test., Personagens, Lugares, Milagres
Modo Rápido com 10 perguntas
As alternativas embaralham a cada rodada para não decorar posição
Ao responder aparece a explicação bíblica com o versículo
3 estrelas de avaliação no final
Melhor pontuação salva no celular
Visual estilo quiz profissional — fundo estrelado azul/dourado
Como colocar no GitHub:
Sobe o arquivo palavra-de-deus.html no mesmo repositório b-blia-palavra- e acessa em https://detiillimichel-max.github.io/b-blia-palavra-/palavra-de-deus.html
